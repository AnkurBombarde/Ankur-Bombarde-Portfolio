Final Year Project

TextEcho: Dynamic Voice Generation is an advanced Text-to-Speech (TTS) system that enables seamless text conversion from images, PDFs, and books using Optical Character Recognition (OCR). Designed as an API, it allows developers to integrate its features into applications effortlessly. Unlike most TTS tools, TextEcho offers customizable voice generation using Tacotron 2 and WaveNet, allowing users to create personalized, natural-sounding speech. Additionally, it supports multiple languages, enhancing accessibility and usability for diverse users.

This project is patented.

Feature

  • Customizable Voice Output - Generates speech that replicates the user's unique voice characteristics while maintaining clarity and efficiency.
  • Multi-Source Text Processing – Extracts and processes text from various sources, including articles, papers, notes, images, and PDFs.
  • Personalized & Realistic Speech – Uses deep learning neural networks (Tacotron 2 & WaveNet) for natural, user-specific voice synthesis.
  • Text Processing Module – Cleans and prepares text for synthesis.
  • Voice Cloning Module – Creates a personalized voice model.
  • Speech Synthesis Module – Converts text into speech using the custom voice.
  • Accessibility & Inclusivity – Benefits individuals with reading difficulties or visual impairments by providing an auditory alternative to reading.
  • Real-Time & Efficient Processing – Ensures fast and smooth text-to-speech conversion for seamless user experience.
  • Speech Storage & Download – Allows users to save, replay, and download generated speech for offline access.
  • Multi-Language Support – Enables users from different linguistic backgrounds to access content in their preferred language.
